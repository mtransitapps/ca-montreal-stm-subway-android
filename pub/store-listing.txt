Full description:
BETA: Thanks for your patience while the last features are being added and the last bugs are being fixed.

This application adds Montreal STM subway information to MonTransit BETA.

STM subways serve Montreal, Laval and Longueuil in Quebec, Canada.

The app contains the planned schedule as well as real time service updates from www.stm.info.

Once this application is installed, the MonTransit BETA app will display subway information (schedule...).

This application only has a temporary icon and no screen: download the MonTransit BETA app (free) in the "More ..." section bellow or by following this Google Play link http://goo.gl/pCk5mV

You can install this application on the SD card but it is not recommended.

The information comes from the GTFS file provided by the Société de Transport de Montréal.
http://www.stm.info/en/about/developers

This application is free and open-source:
https://github.com/mtransitapps/ca-montreal-stm-subway-android

This application is not related with the Société de Transport de Montréal.
